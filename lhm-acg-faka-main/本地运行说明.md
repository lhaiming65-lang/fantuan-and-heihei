# ACG 发卡系统 - 本地运行说明

## 方式一：Docker 一键运行（推荐）

**前提**：已安装 [Docker Desktop](https://www.docker.com/products/docker-desktop/)

1. 双击运行 **`docker-run.bat`**
2. 等待容器启动完成后，浏览器访问：**http://localhost:8080/install/step**
3. 安装时数据库信息填写：
   - 数据库主机：`mysql`
   - 数据库名：`acg_faka`
   - 用户名：`root`
   - 密码：`root`
   - 表前缀：`acg_`

---

## 方式二：本地 PHP + MySQL

### 一、环境要求

- **PHP 8.0+**（需启用扩展：gd, curl, pdo, pdo_mysql, json, session, zip, openssl）
- **MySQL 5.6+**（推荐 5.7 或 8.0）

### 二、已完成的配置

1. ✅ `config/store.php` - 应用商店默认配置
2. ✅ `config/database.php` - 本地数据库配置（root/空密码）
3. ✅ `router.php` - PHP 内置服务器路由
4. ✅ `runtime/` - 运行时目录
5. ✅ `run.bat` - 一键启动脚本

### 三、启动步骤

### 1. 创建数据库

在 MySQL 中执行：

```sql
CREATE DATABASE acg_faka CHARACTER SET utf8mb4;
```

### 2. 修改数据库配置（如需要）

如果你的 MySQL 用户名/密码不是 `root`/空，请编辑 `config/database.php` 修改：

- `username` - 数据库用户名
- `password` - 数据库密码
- `database` - 数据库名（默认 acg_faka）

### 3. 启动服务

**方式一：使用 run.bat（推荐）**

双击 `run.bat`，或在项目目录下执行：

```
run.bat
```

**方式二：手动命令**

```bash
php -S localhost:8080 router.php
```

### 4. 访问安装页面

浏览器打开：**http://localhost:8080/install/step**

按提示完成安装（填写管理员账号等），安装完成后会自动跳转到首页。

### 四、常用地址

| 页面     | 地址                          |
|----------|-------------------------------|
| 首页     | http://localhost:8080         |
| 安装     | http://localhost:8080/install/step |
| 后台管理 | http://localhost:8080/admin   |

### 五、使用 Apache/Nginx 时

若使用小皮面板、XAMPP 等，将网站根目录指向项目文件夹即可，`.htaccess` 已配置好伪静态。

Nginx 需添加：

```nginx
location / {
    if (!-e $request_filename){
        rewrite ^(.*)$ /index.php?s=$1 last;
        break;
    }
}
```

### 六、常见问题

**Q: 提示数据库连接失败？**  
A: 检查 MySQL 是否启动，`config/database.php` 中的账号密码是否正确。

**Q: 想重新安装？**  
A: 删除 `kernel/Install/Lock` 文件后，再次访问安装页面。

**Q: 页面 404？**  
A: 使用 `run.bat` 或 `php -S localhost:8080 router.php` 启动，不要直接用 `php -S localhost:8080`。
